二、隐藏Key
1、在项目的根目录下新建一个.env的文件，名称必须是这个
2、安装
pip install python-dotenv

3、 .env 文件
OPENAI_API_KEY='sk-7f4016c1b34a43958d2a289d12f'

4、示例代码
from openai import OpenAI
from dotenv import load_dotenv
# 会默认查找根目录下 .env文件里面的内容
load_dotenv()
client = OpenAI(base_url='https://dashscope.aliyuncs.com/compatible-mode/v1')
user_input = input('请输入你的问题:')
messages = [
{'role': 'user', 'content': user_input},
]
resp = client.chat.completions.create(
model='qwen3-max',
stream=True,
messages=messages
)
for chunk in resp:
print(chunk.choices[0].delta.content, end='')

四、streamlit介绍和安装
1、streamlit是一个用于创建数据应用的开源的python库，它是专为数据科学家和机器学习工程师而设计的。简化了
数据可视化和交互式应用的过程。
2、之前如果需要搭建一个应用的界面，必须先使用html和css或者vue.js来构建界面，再设计接口，前端调用接口，
最终界面才能呈现数据
3、streamlit直接简化了构建界面的过程，只要使用了这个里面提供的相关的组件就可以快速出效果
4、安装
pip install streamlit

5、查看streamlit版本
streamlit --version

6、运行
streamlit run 文件名.py

7、streamlit官方示例
streamlit hello
