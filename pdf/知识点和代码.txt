一、langchain版本
1、目前最新的是1.2.x版本
2、之前版本 0.1、0.2、03，后面就是 1.x版本
3、直接安装langchain，默认就是最新版本的，之前的很多的东西进行改版了，使用不了（会报错，让你采用建议的
语法）
4、langchain查看下载版本
pip show langchain

5、langchain1.x以上版本，要求的 python解析器的版本最低是3.10.x及以上
6、由于大家的电脑系统安装的是不同的版本（专业版、旗舰版、家庭版、企业版、纯净版），执行新版本的
langchain，底层需要一些解析库的支持，而你的电脑上是没有的，因此会报错。大多数问题都是你电脑系统没有
C++生成解析器的库导致的

7、你下载的如果是ARM版本的python解析器，同样的版本，x86就是没有问题，但是ARM版本就是有问题（ARM更
新和优化实际上是没有那么快的，因此同样的版本的python解析器就会出问题）。建议直接安装x86版本，因为这个
版本现在是支持ARM架构的

PydanticOutputParser（老版本）
from langchain_community.chat_models.tongyi import ChatTongyi
from dotenv import load_dotenv
from langchain_core.output_parsers import PydanticOutputParser
from langchain_core.prompts import PromptTemplate, ChatPromptTemplate
# pydantic 是python定义数据类型的库（类型注解）
from pydantic import BaseModel, Field
from typing import List
load_dotenv()
llm = ChatTongyi(model='qwen3-max')
# 定义要输出的数据格式
# ... 表示选填选项
class PersonInfo(BaseModel):
name: str = Field(description='人的姓名')
age: int = Field(description='人的年龄')
skills: List[str] = Field(description='人的技能列表')
# 初始化解析器
parser = PydanticOutputParser(pydantic_object=PersonInfo)
# template = PromptTemplate.from_template('请从下面的简介中提取姓名、年龄和技能。简介内容：{text}')
# prompt = template.format(text='我叫丁致宇，今年30岁了，会python和java还有前端')
template = ChatPromptTemplate.from_messages([
('system', '你是一个信息提取助手。{format_instructions}'),
('human', '请提取信息: {text}')
])
# 准备数据，进行解析（需要把要解析的数据给它让它进行处理）
instructions = parser.get_format_instructions()
# 把提示词模版占位的变量数据进行填充
final_message = template.format_messages(format_instructions=instructions, text='我叫丁致宇，
今年30岁了，会python和java还有前端')
raw_response = llm.invoke(final_message)
# 最终变成想要的数据格式
result = parser.parse(raw_response.content)
print(result)
print('-' * 30)

四、chain链
1、chain（链）是langchain核心的编排工具，本质是任务流水线
2、langchain里面有很多的核心
提示词（prompt）
模型（LLM）
解析器（parse）
...
链的作用就是把这些连接起来

from langchain_core.prompts import PromptTemplate
load_dotenv()
# 场景：实现翻译效果，输入（中文） --- 输出（英文） --- 单任务
"""
场景：多任务
+ 第一个任务：

=> 输入一段文本（可以主题不限定）
=> 把输入的文本进行总结
+ 第二个任务
=> 把第一个任务的总结，翻译成英文
+ 第三个任务
=> 把第二个任务中翻译的总结，变成一个吸引人的英文标题（提取）
"""
llm = ChatTongyi(model='qwen3-max')
# 用户输入的文本信息
user_text = 'Python是一种高级编程语言，语法简洁，易于学习，广泛用于AI、数据分析、Web开发等领域，是目前最受
欢迎语言之一'
# 第一个任务，提示词
sum_template = PromptTemplate.from_template('请总结下面的内容:{text}')
# 第二个任务，提示词
trans_template = PromptTemplate.from_template('请把下面的内容翻译成英文:{text}')
# 第三个任务，提示词
title_template = PromptTemplate.from_template('请把下面的内容修改为一个吸引人英文的标题:{text}')
# 使用chain（链）
# | 管道符
chain = sum_template | llm | trans_template | llm | title_template | llm
# 一键调用
final_result = chain.invoke({'text': user_text})
print(final_result.content)

五、RunnableParallel
1、RunnableParallel是链中的一个方法，可以同时运行多个任务
2、示例代码
from langchain_community.chat_models.tongyi import ChatTongyi
from dotenv import load_dotenv
from langchain_core.prompts import PromptTemplate
load_dotenv()
llm = ChatTongyi(model='qwen3-max')
# 用户输入的文本信息
user_text = 'Python是一种高级编程语言，语法简洁，易于学习，广泛用于AI、数据分析、Web开发等领域，是目前最受
欢迎语言之一'
# 第一个任务，提示词
sum_template = PromptTemplate.from_template('请总结下面的内容:{text}')
# 第二个任务，提示词
trans_template = PromptTemplate.from_template('请把下面的内容翻译成英文:{text}')

# 第三个任务，提示词
title_template = PromptTemplate.from_template('请把下面的内容修改为一个吸引人英文的标题:{text}')
# 同时运行多个任务并拿到每一个任务的结果
from langchain_core.runnables import RunnableParallel
# 使用chain
# 参数要求是一个字典，因为要执行多任务
chain = RunnableParallel({
'sum_result': sum_template | llm,
'trans_result': trans_template | llm,
'title_result': title_template | llm
})
# 拿到结果
result = chain.invoke({'text': user_text})
# 拿到第一任务的结果
print('总结:', result['sum_result'].content)
# 拿到第二任务的结果
print('翻译:', result['trans_result'].content)
# 拿到第三任务的结果
print('标题:', result['title_result'].content)

六、LLMchain基础链
1、LLMchain是链中最简单的一种的，也是其他链的基石，其他链都是对它的扩展
2、需求：根据“主题、受众、字数”三个参数，生成对应的科普短文。
不使用基础链
from langchain_community.chat_models.tongyi import ChatTongyi
from dotenv import load_dotenv
from langchain_core.prompts import PromptTemplate
load_dotenv()
llm = ChatTongyi(model='qwen3-max')
# 需求：根据“主题、受众、字数”三个参数，生成对应的科普短文。
# audience 受众（小学生、中学生、大学生）
# word_count 字数
# topic 主题
template = PromptTemplate.from_template('给{audience}写一篇{word_count}字左右的{topic}科普短文，
语言通俗，避免专业术语，加入简单例子。')
prompt = template.format(audience='中学生', word_count=200, topic='人工智能')
result = llm.invoke(prompt).content
print(result)

使用基础链（代码灵活性更强、可复用、一定程度上减少了代码量）

# 使用基础链
llm = ChatTongyi(model='qwen3-max')
prompt = ChatPromptTemplate.from_messages([
('human', '给{audience}写一篇{word_count}字左右的{topic}科普短文，语言通俗，避免专业术语，加入简
单例子。')
])
# llm 表示使用那个模型
# prompt 表示提示词
# output_key 定义最终输出结果的字段，之前很多默认的都是 content
# verbose=True 可以看到模型执行过程，可以理解为日志
chain = LLMChain(
llm=llm,
prompt=prompt,
output_key='科普短文',
verbose=True
)
result1 = chain.invoke({
