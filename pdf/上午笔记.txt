三、数据持久化
1、问题：定义了一个数值类型的变量，初始值为0，当点击按钮后让其变量值每次+1，发送页面中实际上没有变化的
（只改变了一次）

import streamlit as st
a = 0
click_btn = st.button('加1')
if click_btn:
a += 1
st.write(a)
print(a)

2、原因：当修改代码后，streamlit每次都会自动更新，重新执行python文件，相当于是把代码从头到尾都重新运行
一遍，所以直接修改数据是不会变化的
3、解决方案：数据持久化
st.session_state 可以实现数据持久化

# {} 默认是一个空字典 --- st.session_state = {}
# print(st.session_state)
# 先查看a这个属性是否在st.session_state中存在
if 'a'not in st.session_state:
# 不存在
st.session_state['a'] = 0
click_btn = st.button('加1')
if click_btn:
st.session_state['a'] += 1
st.write(st.session_state['a'])
