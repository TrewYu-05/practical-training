一、AI概述
1、什么是人工智能？--- AI
定义：AI是计算机科学的一个分支，在模拟人类智能（学习、推理、感知、决策），使机器能执行负责的任务（语音
识别、图像分类、自动驾驶）
核心能力：机器学习、深度学习、自然语言处理（NLP）、计算机视觉等技术（OpenCV --- python库）
应用场景：医疗诊断（专有模型，需要模型的训练 --- 幻觉问题 --- RAG --- 阿福）、金融风控、智能客服、自动驾驶
（智驾）
关键特点：AI是底层技术框架、其他的概念（LLM---大语言模型、AIGC --- 生图、生视频）

2、LLM（大语言模型）
定义：LLM是AI的子集，专供的是自然语言处理，通过海量的数据训练模型，能理解、生成人类的语言
现在模型很多都是基于transforer架构的

3、Prompt（提示词）
定义：用户给AI的输入（指令），用于引导模型生成特定的内容
提示词的好与坏决定了AI给你的回复的指令（提示词工程）

4、MCP（模型上下文协议）
定义：为AI与应用程序提供了一个统一的交互协议

5、AGI（通用人工智能）
通用人工智能也被称为强人工智能，是人工智能领域中的终极追求目标
简单来说，它是指拥有和人一样的能思考、学习、并适应和能解决任何领域中的问题（和人是同等）

二、什么是智能体开发
1、所谓的智能体开发，指的是构建一个能自主思考、决策并执行任务的AI程序。这个程序并不是简单的回答你一个问
题，而是能理解你的目标，自己规划步骤、调用外部的工具，最终完成一个复杂的任务
2、智能体步骤（langchain1.2版本）

六、OpenAI规范
1、如果通过python的requests模块调用大模型，过程是非常繁琐的，因此OpenAI出了一套规范，可以让大家便捷的
调用模型接口
2、安装
pip install openai

3、示例代码
from openai import OpenAI
# 初始化客户端（建立连接）
client = OpenAI(
base_url='https://dashscope.aliyuncs.com/compatible-mode/v1',
api_key='sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx',
)
# 发送请求 --- 补全资源
resp = client.chat.completions.create(
model='qwen3-max',
messages=[
{'role': 'user', 'content': '成都新都有什么好玩的？'}
]
)
# 拿到返回结果
print(resp.choices[0].message.content)

七、流式输出
1、流式输出就是打字的效果
2、示例代码
from openai import OpenAI
# 初始化客户端（建立连接）
client = OpenAI(
base_url='https://dashscope.aliyuncs.com/compatible-mode/v1',
api_key='sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx',
)
# 发送请求 --- 补全资源
# stream=True, # 是否开启流式输出
resp = client.chat.completions.create(
model='qwen3-max',
stream=True,
messages=[
{'role': 'user', 'content': '成都新都有什么好玩的？'}
]
)
print(resp)
print('-' * 30)
# 拿到返回结果
for chunk in resp:
print(chunk.choices[0].delta.content, end='')
